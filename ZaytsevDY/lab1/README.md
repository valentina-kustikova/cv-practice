# Практическая работа №1. Обработка изображений с использованием библиотеки OpenCV
## Описание проекта

Данный проект представляет собой библиотеку фильтров для обработки изображений, реализованную с использованием базовых
операций OpenCV на Python. Все функции реализованы без
использования высокоуровневых функций OpenCV, только с применением базовых операций над матрицами.

## 1. Функция изменения разрешения изображения
Алгоритм реализации:

Вычисляются соотношения размеров исходного и нового изображения

Для каждого пикселя нового изображения определяется соответствующий пиксель в исходном изображении методом ближайшего соседа

Происходит прямое копирование значений пикселей

Особенности: Используется простейший метод масштабирования без интерполяции.

## 2. Функция применения фотоэффекта сепии
Алгоритм реализации:

Для каждого пикселя вычисляется интенсивность по формуле: 0.299*R + 0.587*G + 0.114*B

Новые значения каналов рассчитываются:

R = intensity + 2*k

G = intensity + 0.5*k

B = intensity - k

Значения ограничиваются диапазоном [0, 255]

Особенности: Параметр k регулирует интенсивность эффекта.

## 3. Функция применения фотоэффекта виньетки
Алгоритм реализации:

Вычисляется центр изображения

Для каждого пикселя рассчитывается расстояние до центра

Определяется максимальное возможное расстояние в изображении

Коэффициент затемнения вычисляется как: 1 - strength * (distance / max_distance)

Значения пикселей умножаются на коэффициент затемнения

Особенности: Параметр strength регулирует силу эффекта виньетирования.

## 4. Функция пикселизации заданной прямоугольной области
Алгоритм реализации:

Выделяется указанная прямоугольная область

Область разбивается на блоки размером pixel_size

Для каждого блока вычисляется среднее значение цвета

Все пиксели в блоке заменяются на среднее значение

Особенности: Работает с произвольной прямоугольной областью любого размера.

## 5. Функция наложения прямоугольной одноцветной рамки
Алгоритм реализации:

Создается копия исходного изображения

Заполняются верхняя, нижняя, левая и правая полосы указанной ширины заданным цветом

Особенности: Цвет задается в формате BGR.

## 6. Функция наложения фигурной одноцветной рамки
Алгоритм реализации:
Поддерживаются 5 типов рамок:

Solid - сплошная рамка (аналогично функции 5)

Dashed - пунктирная рамка с чередованием штрихов и пробелов

Double - двойная рамка с внутренним и внешним контуром

Wave - волнистая рамка, создаваемая с помощью синусоидальной функции

Photo_frame - наложение готовой рамки из файла photo_frame.png

Особенности: Для паттерна "photo_frame" изображение автоматически масштабируется под размер рамки.
## 7. Функция наложения эффекта бликов объектива камеры
Алгоритм реализации:

Загружается изображение с бликами на черном фоне (flare.png)

Изображение блика изменяется под размер исходного изображения

Применяется режим наложения "Screen" (Экран) по формуле:

новый_цвет = 255 - ((255 - исходный_цвет) * (255 - цвет_блика) / 255)
Формула применяется отдельно для каждого цветового канала (R, G, B)

Особенности: Черный фон в изображении блика игнорируется, накладываются только светлые области.
## 8. Функция наложения текстуры акварельной бумаги
Загружается текстура акварельной бумаги (water.jpg)

Текстура изменяется под размер исходного изображения

Для каждого пикселя вычисляется яркость текстуры бумаги

Яркость изображения корректируется в зависимости от яркости текстуры:

коэффициент_яркости = 0.6 + 0.4 * яркость_текстуры

новый_цвет = исходный_цвет * коэффициент_яркости

Особенности: Более темные области текстуры сильнее затемняют изображение, светлые - оставляют почти без изменений.
