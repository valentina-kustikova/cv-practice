# Практическая работа №1. Обработка изображений с использованием библиотеки OpenCV

## Цель работы
Реализовать набор фильтров для обработки изображений с использованием **базовых операций над матрицами**, без применения высокоуровневых функций OpenCV.  


---

## Список реализованных фильтров

### 1. Изменение разрешения изображения
**Файл:** `resolution.py`  
**Описание:** выполняет масштабирование изображения методом ближайшего соседа.  
Реализация использует прямое обращение к пикселям с пересчётом координат.  
**Тип обработки:** попиксельная (допустимо по заданию).

---

### 2. Фотоэффект "Сепия"
**Файл:** `sepia.py`  
**Описание:** реализован через матричное преобразование каждого пикселя с использованием матрицы сепии.  
**Тип обработки:** векторное умножение без циклов.

---

### 3. Эффект виньетки
**Файл:** `vignette.py`  
**Описание:** вычисляется маска затемнения по расстоянию от центра кадра, умножается на исходное изображение.  
Используются координатные сетки (`np.indices`) и операции над массивами.  
**Тип обработки:** полностью векторизованная.

---

### 4. Пикселизация заданной области
**Файл:** `pixelation.py`  
**Описание:** выбранная область делится на блоки, для каждого блока вычисляется средний цвет.  
**Тип обработки:** блочная, с использованием циклов (допустимо по заданию).

---

### 5. Простая рамка
**Файл:** `simple_border.py`  
**Описание:** создаётся новая матрица большего размера, изображение вставляется в центр, а края заполняются цветом рамки.  
**Тип обработки:** матричная.

---

### 6. Фигурная рамка
**Файл:** `shaped_border.py`  
**Описание:** формируется маска границ на основе синусоиды или зигзага.  
Маска заполняется построчно.  
**Тип обработки:** векторно-построчная.

---

### 7. Эффект бликов объектива
**Файл:** `lens_flare.py`  
**Описание:** на изображение накладывается текстура блика, масштабированная и позиционированная с помощью NumPy-операций.  
Реализовано векторное масштабирование без циклов.  
**Тип обработки:** полностью векторизованная.

---

### 8. Текстура акварельной бумаги
**Файл:** `watercolor_paper.py`  
**Описание:** текстура бумаги нормализуется и умножается на изображение, создавая эффект бумажной поверхности.  
Используется только векторная арифметика.  
**Тип обработки:** векторная.  


---

## Интерфейс
**Файл:** `main.py`  
Программа содержит графический интерфейс на **Tkinter**, позволяющий:
- загружать изображение;
- выбирать фильтр;
- настраивать параметры;
- отображать исходное и обработанное изображение.

---

## Вывод
В ходе работы была разработана библиотека фильтров, использующая только базовые операции над изображениями.  
Все фильтры реализованы с использованием **NumPy** и соответствуют требованиям задания.  
Попиксельная обработка присутствует только в задачах, где она необходима (масштабирование и пикселизация).
