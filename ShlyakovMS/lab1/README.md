## "Практическая работа №1. Обработка изображений с использованием библиотеки OpenCV"

## Описание программы

Программа предназначена для обработки изображений с помощью различных фильтров и масок.  


## Реализованные функции

### 1. Изменение разрешения изображения — `Change_Res(img, nx, ny)`

Изменяет размер изображения на заданные параметры ширины и высоты.  
Масштабирование выполняется через линейную интерполяцию по осям X и Y, без потери данных.

### 2. Фотоэффект сепии — `Sep(img)`

Преобразует изображение в тёплые тона.  
Формулы преобразования:

tr = 0.393 * r + 0.769 * g + 0.189 * b  
tg = 0.349 * r + 0.686 * g + 0.168 * b  
tb = 0.272 * r + 0.534 * g + 0.131 * b  


### 3. Фотоэффект виньетки — `Vinyetka(img)`

Затемняет края изображения, оставляя центр ярким.  
Коэффициент яркости каждого пикселя зависит от расстояния до центра:

coef = 1 - (dist / diag)**2


### 4. Пикселизация области изображения — `Pixels(image, points)`

Позволяет выбрать прямоугольную область изображения (двумя кликами мыши),  
внутри которой выполняется пикселизация — каждый блок `20×20` заменяется средним цветом.


### 5. Прямоугольная рамка — `Rec_Frame(image, thick)`

Добавляет однотонную рамку по краям изображения.  
Цвет рамки: `[120, 50, 200]`.  
Толщина задаётся пользователем через всплывающее.

### 6. Фигурная рамка — `Frames(image, n)`

Функция **Frames** добавляет к изображению рамку, выбранную по номеру `n_frame` из трёх файлов — `Frame1.png`, `Frame2.png`, `Frame3.png`.
Загруженная рамка масштабируется до размеров изображения с помощью ранее написанной функции `Change_Res`, чтобы точно совпадать по ширине и высоте.
Затем пиксели рамки, отличные от белого цвета (`[255, 255, 255]`), накладываются поверх исходного изображения.

### 7. Эффект блика камеры — `Camera_Light(image)`

Функция **Camera_Light** накладывает на верхнюю левую часть изображения эффект яркого блика.
Для этого используется картинка `blind.png`, которая масштабируется до половины размеров исходного изображения с помощью ранее реализованной функции `Change_Res`.
Пиксели, отличные от белого, смешиваются с исходным изображением в пропорции 0.8:0.2.

### 8. Эффект водяной бумаги — `Aqua_Paper(image)`

Загружается текстура `aqua.png`, масштабируется под размер исходного изображения и накладывается поверх.
Результат получается путём смешивания изображений в пропорции `0.3 * aqua + 0.7 * image`.


## Интерфейс программы

Программа имеет графический интерфейс на **Tkinter**, включающий:

- **Выбор изображения** с компьютера.  
- **Просмотр выбранного изображения** в уменьшенном виде (масштабируется, чтобы поместиться в окно).  
- **Применение фильтров и эффектов** через кнопки интерфейса.
