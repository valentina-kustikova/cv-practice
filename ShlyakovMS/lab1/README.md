
## "Практическая работа №1. Обработка изображений с использованием библиотеки OpenCV"

### 1. Изменение разрешения изображения — `Change_Res(img, nx, ny)`

Изменяет разрешение исходного изображения `img` до размеров `nx, ny`.  
Каждый пиксель нового изображения вычисляется по формуле:  
`result[y][x] = img[y*h//ny][x*w//nx]`, где `h` и `w` — исходная высота и ширина.  
Результат сохраняется как новое изображение `result`.

### 2. Фотоэффект сепии — `Sep(img)`
 
Для каждого пикселя `(b, g, r)` вычисляются новые значения по формуле:

- `r = min(r * 0.393 + g * 0.769 + b * 0.189, 255)`  
- `g = min(r * 0.349 + g * 0.686 + b * 0.168, 255)`  
- `b = min(r * 0.272 + g * 0.534 + b * 0.131, 255)`

Функция возвращает новое изображение с эффектом сепии.

### 3. Фотоэффект виньетки — `Vinyetka(img)`

Затемняет края изображения, оставляя центр ярким.  
Для каждого пикселя вычисляется расстояние `dist` до центра и коэффициент яркости:  
`coef = 1 - (dist / diag)**2`.  
Чем ближе пиксель к краю, тем темнее он становится.

### 4. Пикселизация области изображения — `Pixels(image, points)`

Выполняет пикселизацию заданной области изображения.  
Область задаётся двумя точками `points[0]` и `points[1]`, которые пользователь выбирает кликом мыши на изображении.  
Каждый блок размером `block_size, block_size` заменяется на средний цвет `mean_color`.  
Обработка применяется только внутри выделенной области.

### 5. Прямоугольная рамка — `Rec_Frame(image, thick)`

Добавляет рамку по краям изображения толщиной `thick`. Толщина задается пользователем в графическом интерфейсе.  
Рамка окрашивается в цвет `[120, 50, 200]`.  
Цвет задаётся прямым присвоением пикселей по границам изображения.

### 6. Фигурная рамка — `Frames(image, n_frame)`

Добавляет декоративную рамку из заранее загруженных файлов.  
Тип рамки задаётся индексом `n_frame`, который выбирает один из файлов:

- `Frame1.png`  
- `Frame2.png`  
- `Frame3.png`

Рамка масштабируется под размер изображения функцией `Change_Res` и накладывается на исходное изображение при помощи пропуска фоновых пикселей в изображении рамки.

### 7. Эффект бликов камеры — `Camera_Light(image)`

Создаёт эффект бликов при помощи текстуры `blind.png`.  
Изображение блика уменьшается в два раза и накладывается по формуле:  
`image[y][x] = (0.8 * light[y][x] + 0.2 * image[y][x])`.  
Наложение применяется на определенную область исходного изображения, выбранную заранее при разработке.

### 8. Эффект акварельной бумаги — `Aqua_Paper(image)`

Добавляет текстуру акварельной бумаги из файла `aqua.png`.  
Текстура масштабируется под размеры исходного изображения и накладывается с коэффициентами:  
`image[y][x] = (0.3 * aqua[y][x] + 0.7 * image[y][x])`.  
Создаёт эффект рисунка на акварельной бумеге исходного изображения.

## Интерфейс программы

Программа имеет графический интерфейс на **Tkinter**, включающий:

- **Выбор изображения** с компьютера.  
- **Просмотр выбранного изображения** в уменьшенном виде (масштабируется, чтобы поместиться в окно).  
- **Применение фильтров и эффектов** через кнопки интерфейса.
