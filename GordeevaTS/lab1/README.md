# Практическая работа №1. Обработка изображений с использованием библиотеки OpenCV
---

## 1. Функция `resize_image(image_path, width, height, scale)`
**Масштабирует изображение с загрузкой из файла**

**Вход:**

image_path — путь к исходному изображению \
width, height — новая ширина и высота в пикселях  \
scale — коэффициент масштабирования

**Алгоритм:**

Загружает изображение, вычисляет новые размеры на основе параметров.
Создает равномерно распределенные индексы через np.linspace и meshgrid.
Формирует новое изображение путем выборки пикселей по индексам.

---

## 2. Функция `apply_sepia(image, intensity)`
**Применяет фильтр сепия к изображению**

**Вход:**

image — исходное изображение \
intensity — интенсивность эффекта (0.0-1.0)

**Алгоритм:**

Создает матрицу преобразования цветов в сепию.
Умножает каждый пиксель на матрицу через векторные операции.
При intensity < 1 смешивает результат с оригиналом.

---

## 3. Функция `apply_vignette(image, intensity)`
**Создает эффект виньетки (затемнения по краям)**

**Вход:**

image — исходное изображение \
intensity — интенсивность эффекта (0.0-1.0)

**Алгоритм:**

Вычисляет радиальное расстояние от центра каждого пикселя.
Создает маску с круговым затуханием яркости.
Умножает изображение на маску для создания эффекта.

---

## 4. Функция `apply_pixelation(image, x, y, width, height, pixel_size)`
**Пикселизирует заданную прямоугольную область**

**Вход:**

image — исходное изображение \
x, y — координаты области \
width, height — размеры области   \
pixel_size — размер пикселя эффекта

**Алгоритм:**

Выделяет область, уменьшает ее в pixel_size раз через resize_image.
Увеличивает обратно до исходного размера через resize_image.
Заменяет область в исходном изображении.

---

## 5. Функция `apply_frame(image, frame_width, frame_color)`
**Добавляет прямоугольную рамку вокруг изображения**

**Вход:**

image — исходное изображение \
frame_width — ширина рамки в пикселях \
frame_color — цвет рамки (BGR)

**Алгоритм:**

Создает изображение, заполненное цветом рамки.
Вставляет исходное изображение в центр через индексацию.

---

## 6. Функция `apply_figure_frame(image, frame_path)`
**Накладывает фигурную рамку из файла**

**Вход:**

image — исходное изображение \
frame_path — путь к файлу с рамкой

**Алгоритм:**

Загружает и масштабирует рамку под размер изображения.
Создает альфа-маску на основе яркости рамки.
Области с низкой яркостью (<=0.9) становятся непрозрачными для рамки.
Выполняет альфа-смешивание изображения с рамкой.

---

## 7. Функция `apply_lens_flare(image, flare_path, intensity, position)`
**Добавляет эффект бликов объектива**

**Вход:**

image — исходное изображение  \
flare_path — путь к файлу с бликом \
intensity — интенсивность эффекта \
position — позиция блика (опционально)

**Алгоритм:**

Загружает изображение блика, создает альфа-маску из яркости.
Позиционирует блик на изображении (по умолчанию - правый верхний угол).
Применяет screen blending с учетом альфа-канала.

---

## 8. Функция `watercolor_texture(image, intensity)`
**Накладывает текстуру акварельной бумаги**

**Вход:**

image — исходное изображение \
intensity — интенсивность текстуры

**Алгоритм:**

Загружает текстуру бумаги по фиксированному пути.
Масштабирует текстуру под размер изображения.
Создает маску из инвертированной яркости текстуры.
Смешивает изображение с текстурой через маску.
