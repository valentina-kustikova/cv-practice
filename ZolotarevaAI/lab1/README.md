# Практическая работа №1. Обработка изображений с использованием библиотеки OpenCV

## Цель работы
Изучить базовые операции обработки изображений на Python с использованием библиотеки OpenCV.

## Алгоритмы реализованных функций

1. **resize_image(image, scale=0.5)**  
   Изменяет разрешение изображения с помощью **интерполяции по ближайшему соседу**. Создаются индексы `y_idx` и `x_idx` для нового размера изображения с использованием `np.linspace`. Формируется новое изображение путем выбора пикселей исходного изображения по рассчитанным индексам. 

2. **apply_sepia(image)**  
   Применяет эффект сепии через матричное преобразование цветовых каналов. Каждый пиксель умножается на матрицу по формуле(с помощью каналов пикселя).

3. **apply_vignette(image, strength)**  
   Создает эффект виньетки — затемнение краёв изображения с помощью радиальной маски.

4. **pixelate_region(image, x, y, w, h, pixel_size)**  
   Взятие области в изображении. Выбранная область уменьшается через `cv2.resize` до размера `(w // pixel_size, h // pixel_size)`. После этого увеличивается обратно до исходного размера с `cv2.INTER_NEAREST` для пикселизации. Зона заменяется в исходном изображении.

5. **add_rect_frame(image, color, thickness)**  
   Создается маска для рамки, выбранные пиксели заменяются заданным цветом.

6. **add_shape_frame(image, color, thickness, shape)**  
   Создается макса для рамки, выбранные пиксели заменяются текстурой через смешивание с исходным изображением через альфа-канал.

7. **add_lens_flare(image, intensity)**  
   Текстуру блика масштабируем под размер изображение. Смешиванием  с заданным коэффициентом и исходным изображением.

8. **add_paper_texture(image, strength)**  
   Текстуру бумаги масштабируем под размер изображения. Смешиваем с заданным коэффициентом. Используется "альфа-смешивание", центр изображения остаётся неизменным, текстура накладывается равномерно.
