# Практическая работа №3. Классификация изображений с использованием библиотеки OpenCV

## Описание проекта
Разработано приложение для классификации изображений известных достопримечательностей Нижнего Новгорода:
1. Нижегородский Кремль
2. Дворец труда  
3. Архангельский собор

## Реализованные алгоритмы

### 1. Алгоритм "Мешок слов" (Bag of Visual Words)
**Особенности реализации:**
- Поддержка различных детекторов: SIFT, ORB, AKAZE
- Автоматическое построение словаря визуальных слов
- Нормализация гистограмм частот
- Использование SVM/RandomForest для классификации
- Визуализация ключевых точек

**Алгоритм работы:**
1. Извлечение локальных признаков со всех обучающих изображений
2. Кластеризация дескрипторов методом K-means
3. Построение гистограмм для каждого изображения
4. Обучение классификатора на полученных признаках

### 2. Сверточная нейронная сеть (CNN) с Transfer Learning
**Особенности реализации:**
- Использование предобученных моделей: VGG16, MobileNetV2, ResNet50
- Заморозка базовых слоев для использования обученных признаков
- Добавление пользовательских полносвязных слоев
- Аугментация данных для улучшения обобщения
- Ранняя остановка и уменьшение скорости обучения

**Алгоритм работы:**
1. Загрузка предобученной модели на ImageNet
2. Замена классификатора на свои слои
3. Дообучение на датасете достопримечательностей
4. Классификация новых изображений
