# Практическая работа №1. Обработка изображений с использованием библиотеки OpenCV

## Задача
Разработать библиотеку фильтров с помощью базовых операций над изображениями в OpenCV.

## Реализованные фильтры

### 1. Изменение разрешения изображения
**Алгоритм:** Метод ближайшего соседа. Для каждого пикселя результирующего изображения вычисляется соответствующий пиксель в исходном изображении.

### 2. Фотоэффект сепии
**Алгоритм:** Матричное преобразование цветов с использованием сепийной матрицы. Интенсивность эффекта регулируется параметром.

### 3. Фотоэффект виньетки
**Алгоритм:** Создание маски затемнения, интенсивность которого уменьшается от краев к центру изображения.

### 4. Пикселизация области
**Алгоритм:** Разделение выбранной области на блоки и замена каждого блока средним значением цвета. Поддерживает интерактивный выбор области.

### 5. Прямоугольная рамка
**Алгоритм:** Добавление рамки заданной ширины и цвета по краям изображения.

### 6. Фигурная рамка
**Алгоритм:** Создание декоративных рамок трех типов:
- **wave:** Волнистая рамка
- **zigzag:** Зигзагообразная рамка  
- **triangle:** Треугольная рамка

### 7. Эффект бликов объектива
**Алгоритм:** Наложение готовой текстуры блика на изображение. Текстура масштабируется до размера, пропорционального исходному изображению, и позиционируется в заданных координатах с регулируемой интенсивностью наложения.

### 8. Текстура акварельной бумаги
**Алгоритм:** Наложение текстуры акварельной бумаги с использованием маски, созданной на основе яркости текстуры. Маска усиливается в зависимости от параметра интенсивности, после чего выполняется смешивание исходного изображения с текстурой.
