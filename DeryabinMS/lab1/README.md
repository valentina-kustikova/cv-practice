# Практическая работа №1. Обработка изображений с использованием библиотеки OpenCV

## Задача
Разработать библиотеку фильтров с помощью базовых операций над изображениями в OpenCV.

## Реализованные фильтры

### 1. Функция изменения разрешения изображения
**Алгоритм:** Метод ближайшего соседа. Для каждого пикселя результирующего изображения вычисляется соответствующий пиксель в исходном изображении с учетом коэффициента масштабирования.

### 2. Функция применения фотоэффекта сепии к изображению
**Алгоритм:** Преобразование цветов по формулам:
- B = 0.272 * R + 0.534 * G + 0.131 * B
- G = 0.349 * R + 0.686 * G + 0.168 * B  
- R = 0.393 * R + 0.769 * G + 0.189 * B
Значения ограничиваются диапазоном [0, 255].

### 3. Функция применения фотоэффекта виньетки к изображению
**Алгоритм:** Создание маски затемнения с использованием Гауссова ядра. Интенсивность затемнения уменьшается от краев к центру изображения.

### 4. Функция пикселизации заданной прямоугольной области изображения
**Алгоритм:** Разделение выбранной области на блоки заданного размера. Для каждого блока вычисляется среднее значение цвета, которым заполняется весь блок.

### 5. Функция наложения прямоугольной одноцветной рамки заданной ширины по краям изображения
**Алгоритм:** Определение границ рамки и заливка соответствующих областей заданным цветом.

### 6. Функция наложения фигурной одноцветной рамки по краям изображения
**Алгоритм:** Реализованы три типа фигурных рамок:
- **wave:** Волнистая рамка с синусоидальным паттерном
- **zigzag:** Зигзагообразная рамка  
- **triangle:** Треугольная рамка с градиентом

### 7. Функция наложения эффекта бликов объектива камеры
**Алгоритм:** Создание нескольких перекрывающихся круговых областей с повышенной ярностью. Интенсивность уменьшается от центра к краям каждого блика.

### 8. Функция наложения текстуры акварельной бумаги
**Алгоритм:** Генерация шумовой текстуры и ее наложение на изображение с последующим легким размытием для имитации характерного для акварели эффекта.

## Использование
Программа запускается из командной строки с параметрами:
- `-i, --image`: путь к исходному изображению 
- `-o, --output`: имя выходного файла
- `-m, --mode`: режим обработки
- Остальные параметры зависят от выбранного режима

Примеры команд:
```bash
python script.py -i input.jpg -m sepia -o output.jpg
python script.py -i input.jpg -m resize -c 0.5
python script.py -i input.jpg -m pixel -x 100 -y 100 -w 200 -h 150 -b 10
