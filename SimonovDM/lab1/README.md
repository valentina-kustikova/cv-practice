# Практическая работа №1. Обработка изображений с использованием библиотеки OpenCV

## 1. Функция изменения разрешения изображения

**Входные параметры**
 - изображение;
 - ширина 
 - высота
 - коэффициент масштабирования

**Алгоритм**
 - Определяются текущие размеры изображения.
 - Если не указано ни одно из параметров (ширина, высота, коэффициент), возвращается исходное изображение.
 - Если задан коэффициент масштабирования, новые размеры вычисляются как произведение исходных на этот коэффициент.
 - Если заданы ширина и/или высота, вычисляется пропорциональный размер для сохранения соотношения сторон.
 - С помощью координатных сеток np.meshgrid создаются индексы для выборки пикселей из исходного изображения.
 - Новое изображение формируется выборкой соответствующих пикселей (эффект ближайшего соседа).

## 2. Функция применения фотоэффекта «сепия»

**Входные параметры**
 - изображение
 - интенсивность эффекта (от 0 до 1).

**Алгоритм**
 - Для каждого пикселя RGB рассчитываются новые значения каналов по формулам эффекта сепии.
 - Каждый канал формируется линейной комбинацией исходных значений, умноженных на заданные коэффициенты.
 - Результаты обрезаются до диапазона [0, 255] с помощью функции np.clip.

## 3. Функция применения фотоэффекта «виньетка»

**Входные параметры**
 - изображение 
 - интенсивность эффекта 
 - радиус виньетки.

**Алгоритм**
 - Вычисляются координатные сетки по осям x и y.
 - Определяется центр виньетки (по центру изображения).
 - Для каждого пикселя вычисляется расстояние до центра и создаётся радиальная маска с плавным затуханием.
 - Маска нормализуется и масштабируется по интенсивности.
 - Изображение умножается на маску, из-за чего края затемняются.

## 4. Функция пикселизации области изображения

**Входные параметры**
 - изображение 
 - координаты прямоугольной области; 
 - размер блока пикселизации.

**Алгоритм**
 - Область выделяется по указанным координатам.
 - Изображение разбивается на блоки заданного размера.
 - Для каждого блока вычисляется средний цвет и заменяет все пиксели внутри этого блока.

## 5. Функция наложения одноцветной прямоугольной рамки

**Входные параметры**
 - изображение 
 - ширина рамки 
 - цвет (R, G, B).

**Алгоритм**
 - Если ширина не указана — она выбирается равной 1/10 от минимального размера изображения.
 - Первые и последние строки и столбцы массива заменяются на указанный цвет.

## 6. Функция наложения фигурной рамки

**Входные параметры**
 - изображение
 - номер рамки (выбор из списка готовых файлов).

**Алгоритм**
 - Загружается выбранная рамка из файла.
 - При необходимости подгоняется под размер изображения.
 - Каждый пиксель рамки сравнивается с белым цветом — если он не белый, он заменяет соответствующий пиксель изображения.


## 7. Функция наложения бликов объектива

**Входные параметры**
 - изображение
 - положение центра блика
 - интенсивность 
 - коэффициент масштабирования

**Алгоритм**
 - Загружается изображение блика и масштабируется. 
 - Вычисляются координаты вставки блика в основное изображение (с проверкой выхода за границы). 
 - Выделяется область, где должен располагаться блик. 
 - Пиксели этой области суммируются с пикселями блика с учётом интенсивности. 
 - Результат обрезается до диапазона [0, 255].

## 8. Функция наложения текстуры акварельной бумаги

**Входные параметры**
 - изображение 
 - интенсивность наложения текстуры

**Алгоритм**
 - Загружается изображение текстуры и масштабируется под размер исходного изображения.
 - Текстура преобразуется в оттенки серого с помощью функции bgr_to_gray.
 - Полученная матрица яркости нормализуется и преобразуется в маску.
 - Маска умножается на интенсивность, затем смешивается с изображением, ослабляя или усиливая контрастность в зависимости от фактуры бумаги.
 - Результат приводится к диапазону [0, 255].
